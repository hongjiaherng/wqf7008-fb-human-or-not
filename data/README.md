# Facebook Recruiting IV: Human or Robot?

Download data from <https://www.kaggle.com/competitions/facebook-recruiting-iv-human-or-bot/data>.

Put the data in the `data` directory. The directory structure should look like this:

```bash
data/
└── facebook-recruiting-iv-human-or-bot/
    ├── bids.csv
    ├── sample_submission.csv
    ├── test.csv
    └── train.csv
```

Read the full data spec at <https://www.kaggle.com/competitions/facebook-recruiting-iv-human-or-bot/data>.
